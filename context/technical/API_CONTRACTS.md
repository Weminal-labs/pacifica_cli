## MCP Tools Contract

These are the tools the MCP server exposes. Each tool has a defined name, description, input schema, and output format.

### Read Tools (no guardrails needed)

| Tool | Input | Output |
|------|-------|--------|
| `pacifica_get_markets` | `{}` | `Market[]` ŌĆö all markets with price, vol, OI, funding |
| `pacifica_get_ticker` | `{ symbol: string }` | `Ticker` ŌĆö real-time bid/ask/last for one market |
| `pacifica_get_orderbook` | `{ symbol: string, depth?: number }` | `OrderBook` ŌĆö bids and asks |
| `pacifica_get_positions` | `{}` | `Position[]` ŌĆö all open positions with PnL |
| `pacifica_get_account` | `{}` | `Account` ŌĆö balance, margin, PnL |
| `pacifica_get_orders` | `{ symbol?: string }` | `Order[]` ŌĆö open orders |
| `pacifica_funding_rates` | `{ symbol?: string }` | Funding rates for all or one market |
| `pacifica_funding_arb_scan` | `{ min_spread?: number }` | Cross-exchange funding comparison |
| `pacifica_funding_history` | `{ symbol: string, days?: number }` | Historical funding data |
| `pacifica_agent_status` | `{}` | Guardrails config + daily usage + recent actions |
| `pacifica_agent_log` | `{ filter?: string }` | Agent action audit trail |
| `pacifica_trade_journal` | `{ period?: string }` | Trade journal entries |
| `pacifica_pnl_summary` | `{ period?: string }` | PnL summary stats |
| `pacifica_position_heatmap` | `{}` | Text-based position visualization |
| `pacifica_risk_summary` | `{}` | Structured risk data for all positions |
| `pacifica_get_smart_orders` | `{}` | Active smart orders |

### Write Tools (guardrails enforced)

| Tool | Input | Output | Guardrail Checks |
|------|-------|--------|-----------------|
| `pacifica_place_order` | `{ symbol, side, size, type?, price?, leverage?, tp?, sl? }` | `Order` | enabled, action allowed, size limit, spending limit, confirmation threshold |
| `pacifica_cancel_order` | `{ order_id: string }` | Confirmation | enabled, action allowed |
| `pacifica_close_position` | `{ symbol: string }` | `{ pnl, fees, fill_price }` | enabled, action allowed |
| `pacifica_modify_order` | `{ order_id, tp?, sl?, price? }` | Updated `Order` | enabled, action allowed |
| `pacifica_set_trailing_stop` | `{ symbol, distance, distance_type? }` | `SmartOrder` | enabled, action allowed |
| `pacifica_set_partial_tp` | `{ symbol, levels: {price, percent}[] }` | `SmartOrder` | enabled, action allowed |
| `pacifica_cancel_smart_order` | `{ smart_order_id: string }` | Confirmation | enabled, action allowed |
