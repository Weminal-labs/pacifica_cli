# Component Inventory

> Generated by setup wizard on 2026-03-29. Will be populated as components are built.

## Ink Components (TUI)

### Layout
- **StatusBar** — Connection status, last update time, network indicator (testnet/mainnet)
- **SectionHeader** — `═══` divider with title text

### Data Display
- **MarketTable** — Multi-column table for market scanner (auto-refreshing)
- **PositionTable** — Position list with colored PnL
- **OrderTable** — Open orders list
- **HeatmapBar** — Single position risk visualization bar
- **HeatmapView** — Full heatmap with all positions + risk summary
- **FundingTable** — Funding rate comparison table
- **JournalView** — Trade journal daily/weekly view

### Formatting
- **PnlText** — Green/red colored PnL value
- **PriceText** — Formatted price with appropriate decimals
- **PercentText** — Signed percentage with color
- **VolumeText** — K/M abbreviated volume

### Interactive
- **InitWizard** — 5-step onboarding wizard (uses @inquirer/prompts)
- **ConfirmPrompt** — Y/n confirmation for trades

## Non-Ink Output (Pipe-Friendly)

Core commands that need to work with `| grep` and `| jq` use plain `console.log` with chalk formatting, not Ink components. This includes:
- `pacifica trade` (non-interactive output)
- `pacifica orders`
- `pacifica positions` (non-live mode)
- `pacifica journal`
- `pacifica agent log`
