/ WEB DASHBOARD GUIDE

How to use Pacifica Intelligence

The web dashboard is the visual layer on top of the Pacifica CLI. It shows your patterns, market conditions, and signals in real time — and connects directly to your local intelligence server when it's running.

/ 01 OVERVIEW

How everything connects

The dashboard reads from two sources — your local intelligence server and the Pacifica testnet API.

# Data flow

Your machine → pacifica intelligence serve → localhost:4242

↓ if offline

Pacifica testnet API → test-api.pacifica.fi

↓ for market data always

Web dashboard → shows real data or honest empty state

⚡ Intelligence server running

All pages show YOUR patterns, YOUR signals, and live condition matching from your trade history.

ℹ Intelligence server offline

Pages fall back to real testnet market data. Pattern pages show example patterns so the UI is never empty.

⚠ New to Pacifica?

Install the CLI first. Run pacifica init --testnet to connect your wallet. Then start trading.

/ 11 FULL WORKFLOW

The complete trader loop

From first signal to executed trade — the recommended flow using both CLI and web together.

1

Start the intelligence server

This feeds the dashboard with live data from your trade history. Run once, leave it running in the background.

Install CLI →
2

Open the Market Scanner

Check which markets have active pattern signals right now. Orange-tinted rows = pattern firing. Look for high win rate + negative funding.

Open Scanner →
3

Click a pattern for the full breakdown

See which conditions are matching (✓) vs near-match (~) vs not yet (✗). Find the best-matching market in the Live Market Scan.

Browse Patterns →
4

Simulate before you risk

Click "Simulate →" from the pattern card. The side and symbol are pre-filled. Adjust size and leverage, check your liquidation price and P&L scenarios.

Open Simulator →
5

Check what top traders are doing

Cross-reference with the Leaderboard. Are high-rep traders positioned the same way? If yes, the signal is stronger.

Leaderboard →
6

Execute via CLI

Once confident, execute in one command. The CLI records the trade, and the intelligence server learns from it.

7

Set an alert daemon

Don't watch the screen all day. The daemon rings when the next setup fires.

# The commands behind steps 6 + 7

$ pacifica simulate long ETH --size 500 --leverage 3

$ pacifica trade buy ETH 500 --leverage 3

$ pacifica alerts add --asset ETH --condition "funding < -0.05%"

$ pacifica alerts daemon start
